# MultiQC_WES
# Whole Exome Sequencing Project, requires preliminary quality check analysis. Multiqc.Snakefile is the a collection of tools put together using Snakemake.
Tools include
  - FastQC
  - QualiMap
  - Samtools Flagstats
  - GATK Select Variants
  - GATK Variant Eval
  - SNPeff
  - BCFtools
  - MultiQC
